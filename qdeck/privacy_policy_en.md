# Privacy Policy

**Last Updated**: November 18, 2025

## 1. Introduction

Thank you for using Qdeck (hereinafter referred to as "the App"). This Privacy Policy explains what information FRAGMENT LLC (hereinafter referred to as "we" or "the Company") collects, uses, and handles when providing the App.

## 2. Information We Collect

### 2.1 Question Set Data
- Question titles
- Question text
- Answer text
- Question category information
- Question creation and update timestamps
- Learning history (completed flags, review counts, etc.)

### 2.2 Image Data
- Problem images captured by camera
- Image metadata (size, format, etc.)
- Temporary image data for OCR processing

### 2.3 OCR Analysis Data
- Text extraction from images using Google Cloud Vision API
- OCR analysis result text data
- API communication logs (used only for error resolution)

### 2.4 Purchase Information
- In-app purchase history
- Purchase transaction IDs
- Subscription status (via RevenueCat)
- Purchase dates and expiration dates

### 2.5 Technical Information
- Device information (OS, device type)
- App version information
- Error logs and crash reports
- Anonymized usage data via Firebase Analytics

## 3. Purpose of Information Use

We use the collected information for the following purposes:

### 3.1 Providing Core Features
- Question set storage and management
- Automatic text extraction via OCR
- Image capture and display
- Question search and organization
- PDF output functionality
- Providing in-app purchase features

### 3.2 Service Improvement
- App feature enhancement
- Usability improvement
- New feature development
- OCR analysis accuracy improvement (using anonymized data only)

### 3.3 Technical Support
- Issue resolution
- Bug fixes
- Security maintenance
- Performance optimization

## 4. Information Sharing and Disclosure

### 4.1 Third-Party Disclosure
We will not provide personal information to third parties without user consent, except in the following cases:
- When disclosure is required by law
- When necessary to protect life, body, or property
- When necessary to cooperate with national or local government agencies in performing duties prescribed by law

### 4.2 Integration with External Services
The App integrates with the following external services:

#### Google Cloud Vision API
- Text extraction from images (OCR processing)
- Privacy Policy: https://policies.google.com/privacy

#### RevenueCat
- In-app purchase management
- Subscription status management
- Privacy Policy: https://www.revenuecat.com/privacy

#### Firebase Analytics
- Collection of anonymized usage data
- App performance analysis
- Privacy Policy: https://firebase.google.com/support/privacy

#### Apple App Store / Google Play Store
- In-app purchase processing
- Purchase transaction management
- Complies with each store's privacy policy

## 5. Data Storage and Management

### 5.1 Storage Location
- On-device storage: Question set data, image data, user settings (SharedPreferences, local device storage)
- RevenueCat: Purchase history, subscription status
- Firebase: Anonymized analytics data
- Temporary cloud processing: OCR analysis images (deleted immediately after processing)

### 5.2 Retention Period
- Question set data: Until deleted by user
- Image data: Until deleted by user
- Purchase history: Period required by law
- Cache data: Until app uninstallation
- Error logs: Maximum 90 days
- OCR processing images: Deleted immediately after processing

### 5.3 Security Measures
- Encrypted storage of on-device data
- Communication encryption via HTTPS
- Access control based on principle of least privilege
- Regular security audits

## 6. User Rights

### 6.1 Data Access and Modification
Users can perform the following operations from the app's settings screen:
- View, edit, and delete question set data
- Add and delete images
- Change categories and learning status
- View purchase history

### 6.2 Data Deletion
- Uninstalling the app will delete all data on the device
- Deleting a question set individually will also delete associated image data
- The "Delete All Data" feature in settings allows batch deletion of all question set data on the device

### 6.3 Data Portability
Users can obtain their data through the PDF output feature for saved question sets.

## 7. OCR Features and Data Processing

### 7.1 Cloud Processing
- OCR image analysis is performed in the cloud using Google Cloud Vision API
- Images for analysis are temporarily sent to Google's servers for API processing
- After processing, sent image data is immediately deleted
- Google does not use sent images for advertising or other purposes

### 7.2 Accuracy Improvement
- Anonymized analysis result statistics may be used to improve OCR accuracy
- Even in this case, no personally identifiable information is included

## 8. In-App Purchases and Subscriptions

### 8.1 Purchase Information Management
- In-app purchases are processed through RevenueCat and Apple/Google store services
- We only retain purchase transaction IDs and subscription status
- Payment information (credit card information, etc.) is not retained by us and is managed by each store

### 8.2 Use of Purchase History
- Purchase history is used only for providing premium features and support
- Not used for third-party sales or marketing purposes

## 9. Minors

The App does not intentionally collect personal information from users under 13 years of age. If we become aware that personal information has been provided by someone under 13, we will delete it promptly.

## 10. Cookies and Similar Technologies

The App uses the following technologies for storing user settings and app operation:
- SharedPreferences (local storage)
- Firebase Analytics (anonymized analytics data)

These are necessary for providing app functionality and are not used for personally identifiable tracking or advertising purposes.

## 11. International Data Transfer

User data may be stored or processed on the following service servers:
- Google Cloud (OCR processing)
- RevenueCat (purchase management)
- Firebase (analytics data)

These servers are located in data centers that meet appropriate security standards. Data transfer outside Japan may occur, but appropriate protective measures are in place.
