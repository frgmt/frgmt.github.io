# Privacy Policy

**Last Updated**: November 6, 2025

## 1. Introduction

Thank you for using My Recipe (hereinafter referred to as "the App"). This Privacy Policy explains what information FRAGMENT LLC (hereinafter referred to as "we" or "the Company") collects, uses, and handles when providing the App.

## 2. Information We Collect

### 2.1 Authentication Information
The App provides integration with the following authentication services:
- Google Sign In
- Sign in with Apple

Information collected during authentication:
- User ID
- Username
- Email address
- Profile image URL
- Authentication tokens

### 2.2 Recipe Data
- Recipe titles
- Ingredient lists
- Cooking steps
- Recipe category information
- Custom icon settings
- Recipe creation and update timestamps

### 2.3 Image Data
- Uploaded recipe images
- Image metadata (size, format, etc.)
- Image storage URLs

### 2.4 AI Analysis Data
- Recipe information extracted from URLs
- HTML analysis results
- AI processing logs (on-device only, no external transmission)

### 2.5 Technical Information
- Device information (OS, device type)
- App version information
- Error logs and crash reports

## 3. Purpose of Information Use

We use the collected information for the following purposes:

### 3.1 Providing Core Features
- Recipe storage and management
- Automatic recipe extraction via AI
- Image upload and display
- Recipe search and organization
- User authentication and account management

### 3.2 Service Improvement
- App feature enhancement
- Usability improvement
- New feature development
- AI analysis accuracy improvement (using anonymized data only)

### 3.3 Technical Support
- Issue resolution
- Bug fixes
- Security maintenance
- Performance optimization

## 4. Information Sharing and Disclosure

### 4.1 Third-Party Disclosure
We will not provide personal information to third parties without user consent, except in the following cases:
- When disclosure is required by law
- When necessary to protect life, body, or property
- When necessary to cooperate with national or local government agencies in performing duties prescribed by law

### 4.2 Integration with External Services
The App integrates with the following external services:

#### Supabase
- Recipe data and image storage
- User authentication management
- Privacy Policy: https://supabase.com/privacy

#### Google Sign In
- User authentication functionality
- Privacy Policy: https://policies.google.com/privacy

#### Sign in with Apple
- User authentication functionality
- Privacy Policy: https://www.apple.com/legal/privacy/

#### flutter_llama (AI Analysis Engine)
- Automatic recipe information extraction (on-device local processing)
- No data transmission to external servers

## 5. Data Storage and Management

### 5.1 Storage Location
- On-device storage: User settings, cache data (SharedPreferences)
- Supabase: Recipe data, image data, authentication information
- AI analysis: Processed on-device only, no external storage

### 5.2 Retention Period
- Authentication information: Until deleted by user
- Recipe data: Until deleted by user
- Image data: Until deleted by user
- Cache data: Until app uninstallation
- Error logs: Maximum 90 days

### 5.3 Security Measures
- Encrypted storage of authentication tokens
- Communication encryption via HTTPS
- Access control based on principle of least privilege
- Regular security audits

## 6. User Rights

### 6.1 Data Access and Modification
Users can perform the following operations from the app's settings screen:
- View authentication account information
- View, edit, and delete recipe data
- Add and delete images
- Change categories and icons

### 6.2 Data Deletion
- Uninstalling the app will delete all data on the device
- The account deletion feature allows deletion of all data on Supabase
- Unlinking authentication will revoke access rights for that platform

### 6.3 Data Portability
Users will be able to obtain their data through the recipe data export feature (planned for future implementation).

## 7. AI Features and Data Processing

### 7.1 Local Processing
- AI recipe analysis is performed locally on the user's device
- Analysis data is not transmitted to external servers
- The AI model is embedded in the app and works offline

### 7.2 Accuracy Improvement
- Anonymized analysis result statistics may be used to improve AI accuracy
- Even in this case, no personally identifiable information is included

## 8. Minors

The App does not intentionally collect personal information from users under 13 years of age. If we become aware that personal information has been provided by someone under 13, we will delete it promptly.

## 9. Cookies and Similar Technologies

The App uses the following technologies for storing user settings and app operation:
- SharedPreferences (local storage)
- Session tokens (authentication maintenance)

These are necessary for providing app functionality and are not used for tracking or advertising purposes.

## 10. International Data Transfer

User data is stored on Supabase servers. These servers are located in data centers that meet appropriate security standards. Data transfer outside Japan may occur, but appropriate protective measures are in place.

## 11. Changes to Privacy Policy

This Privacy Policy may be updated as necessary. Significant changes will be communicated to users through in-app notifications or website announcements.
